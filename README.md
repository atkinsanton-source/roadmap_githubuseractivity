# roadmap_githubuseractivity

With this github user activity tool, you can easily check what your mates have been doing in the last 30 days.
Using the github API tool, i pull the newest activity and display it right there in your CLI.

# Usage: python github_activity.py <username>

Project idea from: https://roadmap.sh/projects/github-user-activity
